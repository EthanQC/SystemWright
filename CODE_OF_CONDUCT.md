# Code of Conduct

## Our pledge

We want SystemWright to be a welcoming, harassment-free project for everyone,
regardless of experience level, background, or identity.

## Our standards

Examples of behavior that helps:

- Being respectful and constructive in issues, PRs, and reviews.
- Accepting feedback gracefully and giving it kindly.
- Focusing on what is best for the project and its users.

Unacceptable behavior includes harassment, personal attacks, discriminatory
language, and publishing others' private information without consent.

## Scope

This applies to all project spaces — issues, pull requests, discussions — and to
public spaces when an individual is representing the project.

## Enforcement

Report unacceptable behavior to the maintainer via GitHub (see `SECURITY.md` for
private-contact options). Maintainers may remove, edit, or reject contributions
that violate this code, and may temporarily or permanently ban any contributor
for behavior they deem inappropriate.

This Code of Conduct is adapted from the
[Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
